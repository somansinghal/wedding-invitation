
# 💍 ShadiVerse – Royal Wedding Invitation Website

A luxury, cinematic Indian wedding invitation website built with HTML, CSS, and JavaScript.

Live demo https://shadiverse.netlify.app
---

## ✨ Features

- 🕉 Sanskrit intro with typing animation
- 🔔 Temple bell sound effect
- 🎵 Background wedding music
- 👑 Royal cinematic opening page
- 🌸 Floating petals animation
- ✨ Golden particle effects
- 📅 Wedding timeline section
- ❤️ Our Story section
- 🔥 Sacred Agni animation
- 📩 RSVP form
- 💬 WhatsApp confirmation button
- 📱 Fully responsive design

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Animations + Gradients)
- JavaScript (Typing Effects + Audio + Animations)
- Google Fonts (Cinzel & Great Vibes)
